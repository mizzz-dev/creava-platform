# writing-rules-ja

## 必須ルール
- コミットメッセージ: 日本語
- PRタイトル/本文: 日本語
- docs更新: 日本語基調
- branch名: 英小文字 + ハイフン、`codex` と `Claude` を含めない

## 推奨テンプレ
- commit: `docs: skills運用ガイドを追加` のように目的先頭
- PR本文: 概要 / 変更内容 / 確認手順 / 影響範囲 / 破壊的変更
- issue: 背景・再現手順・期待・受け入れ条件を明記

## 禁止・注意
- 長文ポエムより、再現可能な事実を優先
- 実装未確認事項は「未確認」と明記
- READMEと実装がずれる場合はコード実態を正とする
