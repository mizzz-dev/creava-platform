# design-principles

## 本repo向けの審美方針
- 主役はブランド体験（main）
- store/fc は「自然に進める導線」
- テンプレ感の強い generic UI を避ける
- 余白・階層・視線誘導を先に設計し、装飾は後

## 実装前に言語化する項目
1. design context: この画面の役割とCV
2. brand personality: トーン（例: 静か / 信頼 / クリエイティブ）
3. reference / anti-reference: 何を採用し何を避けるか
4. aesthetic direction: 余白・タイポ・色・モーション指針

## 避けること
- EC主役に見える情報密度
- CTA過多でHomeの文脈を破壊
- darkでコントラスト不足
- アニメーション優先で可読性を損なう
